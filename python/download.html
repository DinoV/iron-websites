  • What version should I use?

    IronPython maintains versions with Python 2.5 and 2.6: IronPython 2.0.3 and IronPython 2.6, respectively. If you're not sure which version to use, use the latest 2.6 release.

  • What else do I need?

    IronPython is built on the .NET framework. The latest production version is 3.5 SP1, which is recommended, though IronPython will also run on 3.5, 3.0, and 2.0 SP1.
